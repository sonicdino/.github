# Sonicdino

> Music streaming, without the noise.

## What is this?

**Dinosonic** is an OpenSubsonic server. Host your music library and stream it from anywhere. Works with any [OpenSubsonic](https://opensubsonic.netlify.app/) client.

**Dino** apps are the clients built for mobile, desktop and web. They work with Dinosonic or any other [OpenSubsonic](https://opensubsonic.netlify.app/) server.
